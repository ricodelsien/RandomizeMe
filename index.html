    How to use RandomiseMe

    1. Add projects manually or import a .txt or .csv list.

    2. Click "Let's roll!" to randomly select a single project.

    3. Use "Clear all UFOs" to reset your list.

    Imported lists can be separated by commas, semicolons, line breaks or tabs.
